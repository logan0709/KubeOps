# 脚本说明

### 安装docker

- installDockerByUbuntu.sh
- installDockerByCentos.sh

### 卸载docker

- removeDocker.sh



### 安装K8S

- installK8SByUbuntu.sh

### 卸载K8S

- removeK8S.sh

### 重新初始化K8S

> 比如因更换IP导致无法启动K8S

- reInitK8S.sh

### 尝试重启K8S

> 比如因没有关闭交换内存而无法启动K8S

- restartK8S.sh

### 初始化所在用户的K8S

> 比如使用root安装K8S后,想要在用户kubeops中使用

- initThisUserK8S.sh



### 安装node.js

- installNodeByUbuntu.sh
